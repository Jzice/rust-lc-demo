# 专题：买卖股票

## 简介

股票买卖时机问题是 leetcode 中一类很常见的问题, 题目表述如下:

有一个数组 prices, 代表某支股票在每天的价格.

投资者在每天可以选择买入,卖出,或者不操作来对这支股票进行操作.

需要求的投资者获取最大收益而每天执行操作的序列.

## 解题思路

动态规划+贪心
