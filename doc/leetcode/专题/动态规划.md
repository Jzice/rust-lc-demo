# 专题：动态规划

## 简介

动态规划问题的难点在于对问题进行合理划分，将大问题转变为小问题的组合，形成递推关系，通过归纳法将问题得到解决；

动态规划最关键的地方在于对问题进行正确的抽象和建模，进而建立递推公式；
