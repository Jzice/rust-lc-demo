# 专题：子串和子序列

## 简介

子串和子序列是 leetcode 中最常见的题目类型之一. 都是在一个数组上的问题.

- 子串：原数组中的一部分连续元素集合, 元素的相对顺序不变;
- 子序列：原数组中的一部分元素集合, 元素可以不连续, 但相对顺序不变;
